# Age Teller

## A React web application where the user submit a date

To see the project:
- Clone/download it
- Use ```npm install```
- Use ```npm run start```

## This application includes examples of major features of React including:
- ES6 syntax
- Templating string
- Destructuring assignment
- Component-based architecture
